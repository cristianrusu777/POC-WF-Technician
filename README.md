## View Wireframes Locally

To view the wireframes on your local machine, clone the repository using the following command:

```bash
git clone https://github.com/cristianrusu777/POC-WF-Technician.git
```
## Navigation

Open authentication.html file in a browser.

To view the User wireframe setting, click "Authenticate"

To view the Technician wireframe setting, click "Technician Login"
